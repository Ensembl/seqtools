Navigation

Left mouse button

  • Click on residue to show coordinate.
  • Double click on sequence to fetch.
  • Scrollbars: drag and page.
  • Toolbar buttons:
    • File: Display this text.
    • Edit: Remove many sequences.
    • Colour: Edit colour scheme window.
    • Sort: Sort alphabetically.

    • Middle mouse button

    • In alignment: centre on crosshair.
    • Scrollbars: drag and jump.

    Right mouse button

    • Toolbar menus (file menu elsewhere).

    Keyboard

    • Arrow keys: Move screenfulls.
    • Home: Go to Top.
    • End: Go to Bottom.
    • Insert: Go to Start of line.
    • Delete: Go to End of line.
